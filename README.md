# 🍄 Super Mario Bros - HTML5 Playable Version

A fully playable Super Mario Bros game that runs directly in your browser! No downloads or installations required.

## 🎮 Play Online

**Streamlit Version**: Run `streamlit run streamlit_app.py` and open your browser to play!

**Direct HTML**: Open `index.html` in any modern web browser to play directly.

## 🚀 Quick Start

### Option 1: Streamlit Web App (Recommended)

1. **Install Streamlit:**
```bash
pip install streamlit
```

2. **Run the app:**
```bash
streamlit run streamlit_app.py
```

3. **Open your browser:**
   - Go to `http://localhost:8501`
   - Start playing immediately!

### Option 2: Direct HTML Play

1. **Open the game:**
   - Double-click `index.html`
   - Or drag it into your browser
   - Start playing instantly!

## 🎮 How to Play

### Controls
- **⬅️ Left Arrow**: Move Mario left
- **➡️ Right Arrow**: Move Mario right
- **⬆️ Space**: Make Mario jump
- **Click Buttons**: Use the control buttons below the game

### Objective
1. **Collect Coins**: Gather all golden coins to advance levels
2. **Avoid Enemies**: Don't touch the brown Goombas from the side
3. **Jump on Enemies**: Land on top of enemies to defeat them
4. **Stay on Platforms**: Don't fall off the edges
5. **Complete Levels**: Collect all coins to progress

### Scoring
- **Coin**: 100 points
- **Level Complete**: 500 points
- **Lives**: Start with 3 lives
- **Game Over**: When all lives are lost
